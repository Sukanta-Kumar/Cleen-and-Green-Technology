# Clean and Green Technology

Clean and Green Technology is a full stack web application developed to encourage proper waste management using a reward-based system. Users earn Green Coins for eco-friendly actions and can redeem them for plants and compost.

## About the Project
Plastic waste pollution is a major environmental problem. This project aims to involve citizens in cleanliness and sustainability activities by using technology and rewards.

## Features
- User Login and Signup
- Home Dashboard
- Waste Reporting
- Green Coin System
- Eco-friendly Marketplace
- Checkout and Order History
- Sustainability Courses

## Tech Stack
- Frontend: React
- Backend: Spring Boot
- Database: MySQL

## Project Structure
- greentrack-frontend/ – frontend code
- greentrack-backend/ – backend code
- docs/ – SURE Trust project report and documentation

## How to Run

### Frontend
cd greentrack-frontend/greentrack-frontend  
npm install  
npm start  

### Backend
cd greentrack-backend/backend  
mvn spring-boot:run
